# AI_RDDPanama

Automated road defect detection for Panama using [YOLO11n](https://github.com/ultralytics/ultralytics) trained on the [RDD2022ES](https://www.kaggle.com/datasets/juusos/rdd2022es) dataset.

## Usage

All steps are run through `main.py`.

```bash
# Run the full pipeline (download в†’ train в†’ predict)
python main.py

# Run individual steps
python main.py --steps 1        # Download and format dataset
python main.py --steps 2        # Train model
python main.py --steps 3        # Predict on validation set

# Run a subset of steps in order
python main.py --steps 1 2      # Prepare then train, skip predict

# Prediction options (used when step 3 is included)
python main.py --steps 3 --source path/to/image.jpg
python main.py --steps 3 --source path/to/video.mp4
python main.py --steps 3 --source 0                   # webcam
python main.py --steps 3 --weights path/to/custom.pt
python main.py --steps 3 --conf 0.4 --iou 0.5
python main.py --steps 3 --save-txt --save-conf
```

## Pipeline steps

### Step 1 вЂ” Prepare dataset
Downloads RDD2022ES from KaggleHub and splits it 80/20 into `dataset_yaml/images/train` and `dataset_yaml/images/val`. Generates `data.yaml` automatically from `classes.txt` (or by inferring class IDs from the label files if `classes.txt` is absent). Any images missing a label file get an empty `.txt` placeholder so YOLO does not error out.

### Step 2 вЂ” Train
Downloads `yolo11n.pt` into the project folder if not already present. Trains for 5 epochs at `imgsz=640`, `batch=8`, `workers=4` вЂ” tuned for CPU use. Weights are saved to `runs/detect/train/weights/`.

### Step 3 вЂ” Predict
Loads `runs/detect/train/weights/best.pt` and runs inference on `dataset_yaml/images/val/` by default. Results (annotated images, optionally label `.txt` files) are saved to `runs/detect/predict/`.

---

## Notes

- **GPU**: if you have a CUDA-capable GPU, you can increase `epochs` and `batch` in `pipeline/train.py` for significantly faster training and better results.
- **Paths**: all paths are managed centrally in `filehandler.py`. If you need to rename or relocate any folder, that is the only file to edit.
- **Dataset and weights are git-ignored** вЂ” they are either downloaded automatically or produced by the pipeline, so there is no need to commit them.
